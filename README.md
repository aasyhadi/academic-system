# Academic Management System

Academic Management System adalah aplikasi console berbasis Java untuk mengelola data akademik seperti mahasiswa, dosen, mata kuliah, nilai, dan dashboard ringkas.

## Tech Stack

- Java 21
- Maven
- IntelliJ IDEA
- OOP
- Repository Pattern
- Service Layer
- Validation
- Exception Handling
- Singleton Pattern
- Factory Pattern

## Features

- Login Admin
- Dashboard Akademik
- CRUD Mahasiswa
- CRUD Dosen
- CRUD Mata Kuliah
- CRUD Nilai
- Validasi Input
- Auto Increment ID
- Console Utility
- Input Utility
